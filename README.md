# Strange Dungeon

このゲームは、3Dの不思議のダンジョン(ローグライクダンジョンゲーム)です。

## 操作方法
WASDでそれぞれ前左後右に移動できます。また、Spaceでジャンプができます。
攻撃は左クリックでできます。
階段、回復ポイント、クリアオーブの前でFキーを押すことで、アクションを起こせます。
